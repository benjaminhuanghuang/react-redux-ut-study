# Jest

## Setup
``` 
    npm i -D jest babel-jest
```

## Command
```
$ jest --watch
$ jest --coverage
$ jest --config jest.config.js
npm test -- --watch

# Test single file
jest src/utils/sort.test.js


# Test test suit

```

## Run jest by using npm script
```
    "scripts": {
        "test": "jest"

    }
    $ npm test
```
